# 🚀 End-to-End CI/CD Pipeline using Jenkins, Docker & Maven

## 📌 Project Overview
This project demonstrates a complete CI/CD pipeline implemented on Ubuntu using Jenkins. The pipeline automates the process of building, packaging, and deploying a Java web application using Docker.

---

## 🧰 Tech Stack
- Jenkins (CI/CD)
- GitHub (Version Control)
- Maven (Build Tool)
- Docker (Containerization)
- Tomcat (Application Server)
- Ubuntu Server 

---

## 🔄 CI/CD Workflow

1. Developer pushes code to GitHub
2. Jenkins pulls the latest code
3. Maven builds the application (.war)
4. Docker builds the image
5. Container is deployed automatically
6. Application becomes live
